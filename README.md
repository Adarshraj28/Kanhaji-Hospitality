# Kanhaji Hospitality

Professional institutional food management services across India. Specializing in catering, mess management, corporate canteens, hospital meals, and school dining programs.

## Tech Stack

- **Framework:** Next.js 16
- **Styling:** Tailwind CSS v4
- **Fonts:** Inter + Plus Jakarta Sans
- **Icons:** Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

## Project Structure

- `src/app/` - Page routes
- `src/components/` - UI components, sections, layout
- `src/data/` - Constants and content data
- `public/images/` - Stock photography

## Pages

- `/` - Home
- `/services` - Service catalog
- `/industries` - Industries served
- `/menus` - Menu categories
- `/quality` - Quality & hygiene
- `/gallery` - Photo gallery
- `/about` - Company story
- `/blog` - Articles
- `/case-studies` - Client success stories
- `/clients` - Client list
- `/testimonials` - Testimonials
- `/contact` - Contact form
- `/faq` - FAQ
- `/careers` - Job openings
- `/get-proposal` - Proposal request
